WordPress GDPR - Secure your Comments

== Description ==
A simple plugin which add checkbox and Policy information with URL to all WordPress Posts Comments.
<h3>Features</h3>
<ol>
	<li>Policy Title</li>
	<li>Policy Text</li>
	<li>HTML Supported</li>
	<li>Translation Ready</li>
	<li>Option to show Checkbox</li>
	<li>Customize error message if user checkbox not ticked</li>
</ol>

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can set options from `Settings->WP GDPR` menu
